# Custom Flappy Bird Variant

A unique 2D arcade game built with **Unity** and **C#**, featuring customized Flappy Bird mechanics and completely original hand-drawn artwork, optimized for **Full HD (1920x1080)** resolution.

## Original Art & Custom Code
This project is a deeply personalized variation of the Flappy Bird formula.

* **100% Original Art:** All visual assets—the angel character, the ornate, vine-covered columns, and the soft, cloud-filled background—were **designed and drawn entirely by me**, giving the game a unique aesthetic identity.
* **Modified Code:** While inspired by a tutorial, I have modified some of the core code.

## Key Features
* **Hand-Drawn Visuals:** Custom-designed art assets for all in-game objects.
* **Full HD (1920x1080) Precision:** Mechanics are calibrated for a perfect Full HD aspect ratio.

## Learning & Development Process
The core concepts of this project were learned from a YouTube tutorial. However, I advanced the project significantly by:
* **Rewriting key functions** for personalized mechanics.
* **Designing and integrating all original assets.**
* **Calibrating physics and controls** for a tighter, more polished feel.

## Technical Specs
* **Engine:** Unity 2022.3+
* **Language:** C#
* **Resolution:** Optimized for 16:9 (1920x1080)

## Controls
* **Jump:** `Space Bar` or `Left Mouse Click`.
* **Restart:** Click the "Play Again" button on the Game Over screen.
